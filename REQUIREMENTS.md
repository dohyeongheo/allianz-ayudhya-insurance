# Allianz Ayudhya 보험 랜딩 페이지 프로젝트 요구사항 문서

## 1. 프로젝트 개요

### 1.1 프로젝트 목적
Allianz Ayudhya의 건강보험 및 암보험 상품을 태국 시장에 홍보하기 위한 반응형 웹 랜딩 페이지 구축

### 1.2 프로젝트 범위
- 태국 시장 타겟 온라인 보험 랜딩 페이지
- SEO 최적화 및 SNS 공유 최적화
- 실시간 보험료 견적 기능
- 병원 찾기 기능
- 모바일 최적화 (태국 시장의 모바일 중심 사용자 환경 고려)

### 1.3 주요 이해관계자
- Allianz Ayudhya (클라이언트)
- 태국 내 보험 가입 희망 고객
- 내부 마케팅 팀

---

## 2. 기술 스택 요구사항

### 2.1 프레임워크 및 라이브러리
- **Next.js 14+** (App Router 사용)
- **React 18+**
- **TypeScript** (타입 안정성 보장)
- **Tailwind CSS** (유틸리티 퍼스트 CSS 프레임워크)

### 2.2 UI 컴포넌트 라이브러리
- **Shadcn UI** (Button, Card, Badge, Input 등)
- **Framer Motion** (애니메이션 효과)

### 2.3 폰트
- **Google Fonts**
  - Sarabun: 본문 텍스트 (태국어 가독성 최적화)
  - Kanit: 제목 및 강조 텍스트

### 2.4 개발 도구
- **Git** (버전 관리)
- **ESLint** (코드 품질 관리)
- **PostCSS** (CSS 처리)

## 4. 비기능 요구사항

### 4.1 성능 요구사항
- **페이지 로딩 시간**: 3초 이내 (3G 네트워크 기준)
- **이미지 최적화**: Next.js Image 컴포넌트 사용, lazy loading
- **코드 분할**: Next.js 자동 코드 분할 활용
- **번들 크기 최적화**: 트리 쉐이킹 및 동적 import 활용

### 4.2 반응형 디자인 요구사항
- **모바일 우선 설계** (태국 시장 특성)
- **브레이크포인트**:
  - Mobile: 320px ~ 768px
  - Tablet: 768px ~ 1024px
  - Desktop: 1024px 이상
- **원손 썸 네비게이션**: 모바일 사용자 편의성 고려

### 4.3 접근성 요구사항
- **WCAG 2.1 AA 수준 준수**
- **키보드 네비게이션 지원**
- **스크린 리더 호환성** (ARIA 레이블)
- **색상 대비 비율**: 최소 4.5:1 (텍스트)

### 4.4 브라우저 호환성
- **최신 브라우저 지원**:
  - Chrome (최신 2개 버전)
  - Firefox (최신 2개 버전)
  - Safari (최신 2개 버전)
  - Edge (최신 2개 버전)
- **모바일 브라우저**: iOS Safari, Chrome Mobile

### 4.5 SEO 요구사항
- **메타 태그 최적화**:
  - Title 태그 (태국어 키워드 포함)
  - Meta description (태국어)
  - Open Graph 태그 (Facebook, Line 최적화)
  - Twitter Cards 태그
- **구조화 데이터 (JSON-LD)**:
  - Organization 스키마
  - InsuranceProduct 스키마
  - InsuranceAgency 스키마
- **시맨틱 HTML**: H1, H2 태그에 태국어 키워드 포함
- **이미지 최적화**: 태국어 alt 텍스트
- **언어 설정**: HTML lang="th-TH"

### 4.6 SNS 공유 최적화
- **Line 최적화**:
  - 정사각형 이미지 (1200x1200px) 우선 표시
  - 태국어 공유 메시지: "ด้วย Allianz Ayudhya สุขภาพของคุณจะปลอดภัยเสมอ"
- **Facebook/Twitter 공유**:
  - 표준 Open Graph 이미지 (1200x630px)
  - 브랜드 로고 및 신뢰감 있는 이미지 포함

## 5. 디자인 요구사항

### 5.1 컬러 팔레트

- **강조 컬러**: Light Blue (보조 색상)
- **배경**: White (#FFFFFF)
- **텍스트**:
  - 주요 텍스트: 진한 회색
  - 보조 텍스트: 중간 회색
- **Line 브랜드 컬러**: Line Green (Line 연락 버튼)

### 5.2 타이포그래피
- **본문 폰트**: Sarabun (Google Fonts)
  - 가중치: 300, 400, 500, 600, 700
  - 태국어 가독성 최적화된 자간 및 행간
- **제목 폰트**: Kanit (Google Fonts)
  - 가중치: 400, 500, 600, 700
- **글자 간격**: 태국어 가독성을 위한 적절한 letter-spacing
- **줄 간격**: line-height 1.6 이상

### 5.3 UI 컴포넌트 스타일
- **카드**: rounded-2xl (둥근 모서리)
- **버튼**: font-kanit 사용
- **그림자**: shadow-sm (은은한 그림자)
- **호버 효과**: 부드러운 transition

### 5.4 레이아웃 원칙
- **여백**: 충분한 여백으로 깔끔한 느낌
- **그리드 시스템**: Tailwind CSS 그리드 활용
- **모바일 우선**: 모바일에서 최적화된 레이아웃
