## 이미지 생성 권장 사항

1. **고해상도**: 최소 1200px 너비/높이
2. **압축**: 파일 크기 최적화 (1MB 이하 권장)
3. **텍스트 가독성**: 태국어 및 한국어 텍스트가 명확하게 읽혀야 함
4. **브랜드 일관성**: Allianz Blue (#003781) 색상 사용
5. **이미지 형식**: JPEG (사진), PNG (로고가 있는 경우)
